# vedacomputing-website

Gustavo Cabezal website for blog, tutorials, and project exposure.

Website built on Hugo, built and deployed via github actions to firebase hosting.

---

## Tools Used

- Hugo
- Visual Code
- Github Actions:
  - lowply/deploy-firebase
  - lowply/build-hugo
- Firebase Hosting
- Github API
